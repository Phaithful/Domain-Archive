# 🌐 The Domain Archive

**A centralized platform to securely store, organize, and monitor all your domain names in one place.**

## 🚀 Overview

Managing multiple domains can get chaotic—scattered spreadsheets, missed renewals, and forgotten credentials. **The Domain Archive** solves this by providing a **secure, centralized, and easy-to-use dashboard** to help individuals and organizations take control of their entire domain portfolio.

---

## 🧰 Tech Stack

- **Backend:** PHP
- **Frontend:** HTML, Vanilla CSS, Bootstrap 5
- **Interactivity:** JavaScript, JQuery
- **Database:** MySQL

---

## 📦 Features

- ✅ **Secure Storage:** Keep all your domain information in one safe location.
- 📁 **Categorized Views:** Organize domains by status, registrar, expiry, or category.
- 🕒 **Expiration Tracking:** View domains by expiry date to stay ahead of renewals.
- 📊 **Dashboard Overview:** At-a-glance summary of domain counts, categories, and statuses.
- 🛡️ **Authentication (optional):** Add user login to protect your domain list.
- 📝 **CRUD Operations:** Add, update, view, and delete domain entries.

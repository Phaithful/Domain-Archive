# 🌐 The Domain Archive

**A centralized platform to securely store, organize, and monitor all your domain names in one place.**

---

## 🚀 Overview

Managing multiple domains can get chaotic—scattered spreadsheets, missed renewals, and forgotten credentials.

**The Domain Archive** solves this by offering a **secure, centralized, and user-friendly dashboard** for individuals and organizations to easily manage their entire domain portfolio.

---

## 🧰 Tech Stack

- **Backend:** PHP
- **Frontend:** HTML, Vanilla CSS, Bootstrap 5
- **Interactivity:** JavaScript, jQuery
- **Database:** MySQL

---

## 📦 Features

- 🔐 **Secure Storage** – Keep all your domain information in one safe location.
- 🗂️ **Categorized Views** – Organize domains by status, registrar, expiry, or category.
- ⏰ **Expiration Tracking** – Stay ahead of renewals with expiry monitoring.
- 📊 **Dashboard Overview** – View domain counts, statuses, and categories at a glance.
- 👥 **Authentication (optional)** – Protect your dashboard with user login.
- ✏️ **CRUD Operations** – Easily add, edit, delete, and view domains.

## 🔄 Project Workflow

1. Clone the repository
2. Set up the database using `domain_archive.sql`
3. Configure database credentials in `/config/db.php`
4. Run the project via local server (XAMPP recommended)
5. Log in and start managing your domains!
